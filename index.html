HTML stands for HyperText Markup Language. It's the foundational language for creating web pages. HTML uses tags, which are keywords enclosed in angle brackets (e.g., 'p' for paragraph, 'h1' for heading). These tags provide structure to the content, defining elements like headings, paragraphs, images, lists, and links.

Cascading Style Sheets (CSS) is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML or XML (including XML dialects such as SVG, MathML or XHTML).

Java is a multi-platform, object-oriented, and network-centric language that can be used as a platform in itself. It is a fast, secure, reliable programming language for coding everything from mobile apps and enterprise software to big data applications and server-side technologies.
